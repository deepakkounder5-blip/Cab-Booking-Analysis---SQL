# 🚕 Cab Booking System - SQL Analysis Project


📌 Project Overview
This project involves the design and implementation of a relational database for a Cab Booking System. 
It includes the complete database schema (DDL), sample data insertion (DML), and a variety of complex SQL queries designed to extract business insights regarding customer behavior, driver performance, and revenue trends.



🗃️ Database Schema
The project consists of six primary tables:

Customers: Stores personal details and registration dates.

Drivers: Contains driver information and average ratings.

Cabs: Details about vehicles including model and type (Sedan, SUV, Hatchback).

Bookings: Tracks pickup/dropoff locations and booking statuses (Completed, Canceled, Ongoing).

TripDetails: Records trip specific data such as distance (KM) and fare.

Feedback: Captures customer ratings, comments, and cancellation reasons.



📊 Key Business Queries & Insights
The SQL script provides solutions for several business requirements:

Customer Loyalty: Identifies the top 5 customers with the most completed bookings.

Revenue Analysis: Calculates total revenue over the last 6 months and monthly revenue trends.

Performance Monitoring: Lists drivers with an average rating below 3.0 for quality control.

Operational Trends: Analyzes peak booking days and average wait times by pickup location.

Cancellation Analysis: Provides a breakdown of reasons for trip cancellations.

Trip Categorization: Segments trips into 'Short', 'Medium', and 'Long' distances to analyze revenue by category.



🛠️ Technologies Used
Database: MySQL / PostgreSQL (Relational Database Management System).

Language: SQL (Structured Query Language).



🚀 How to Use
Set up the Database: Execute the CREATE DATABASE CB; and use CB; commands.

Create Tables: Run the DDL scripts to establish the schema.

Insert Data: Run the provided INSERT statements to populate the tables with sample records.

Run Analysis: Execute the analytical queries at the end of the script to generate reports.



📈 Sample Results
High Revenue Routes: Identifies the most frequent pickup and dropoff combinations.

Vehicle Performance: Compares total revenue generated by different vehicle types like Sedans vs. SUVs.
